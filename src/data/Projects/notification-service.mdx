# Overview

A production-ready notification microservice that supports Email, SMS, and Push Notifications using asynchronous processing, retry mechanisms, and reusable notification templates.

## Features

- Email Notifications
- SMS Notifications
- Push Notifications
- Retry Mechanism
- Scheduled Notifications
- Notification Templates
- User Preferences

## Challenges

- Implementing retry logic without duplicate deliveries.
- Designing reusable notification templates.
- Processing notifications asynchronously.

## Highlights

- Reduced notification latency using background workers.
- Designed a reusable template engine.
- Implemented retry logic with exponential backoff.

- Microservice Architecture

- Redis

- Production Logging
